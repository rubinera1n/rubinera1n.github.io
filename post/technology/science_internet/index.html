1. 配置前的准备

  • 域名
  • VPS (虚拟专用服务器)

1.1 如何申请域名？

免费：google search apply free domain name

我自己使用阿里付费域名，后面的 HTTPS 也方便，可配置自己博客域名或 GitHub + Netlify 搞静态文件管理都很方便。

阿里申请满简单的，可以根据自己需求，申请一个域名。官方链结

1.2 申请一台vps

关于 VPS 网上很多选择，例如：AWS、GCP、Vultr、搬瓦工、DigitalOcean、Linode…等等，货比三家看哪个适合你。

如果你有 visa / mastercard 信用卡，可以直接去申请Google Cloud Platform，只要信用卡验证成功，就送你 300 美金/年，如果不是重度科学使用者，CP 值高挺推荐，300 美金可以用很久。

我个人使用搬瓦工（没有收广告费），国内可以使用支付宝付款，很早期就购买，我的套餐都绝版了，所以只能续费维持。

VPS 的系统推荐 CentOS 7 / Ubuntu 16,18，大部分网上都有这两系统教程。

这部份需要自己申请 SSL 证书，下载 Nginx + v2ray，自己配置 Nginx 和 v2ray 配置文件。

申请SSL

阿里查询产品与服务，搜索 SSL证书

点击购买证书

直接购买免费一年

购买成功后，在控制台会看到刚刚购买DigiCert 免费版 SSL，点击证书申请

输入自己的域名，点击下一步

申请完，针对你使用web server，下载对应密钥（nginx）

server {
    listen 443 ssl;
    # 阿里上面的域名
    server_name {自己配置的域名};
    # 这边需要提前把前面SSL下载的密钥scp到服务器，路径自己配置
    ssl_certificate      /etc/nginx/key/{pem文件};
    ssl_certificate_key  /etc/nginx/key/{key文件};
    ssl_protocols         TLSv1 TLSv1.1 TLSv1.2;
    ssl_ciphers           HIGH:!aNULL:!MD5;
    location /event {
            proxy_redirect off;
            proxy_pass http://127.0.0.1:10000;
            proxy_http_version 1.1;
            proxy_set_header Upgrade $http_upgrade;
            proxy_set_header Connection "upgrade";
            proxy_set_header Host $http_host;

    # Show realip in v2ray access.log
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
